# AI Learning Repository

זה מאגר ללימוד בינה מלאכותית עם דוגמאות ופרויקטים קטנים שממחישים רעיונות מעשיים בתחום ה-AI ו-LLM.

## מה נמצא כאן?

- Text to Commands — פרויקט שמתרגם טקסט חופשי לפקודות מובנות באמצעות Prompt Engineering ו-LLMs.
- RAG — ניסויים עם Retrieval-Augmented Generation.

## מטרת המאגר

המאגר נועד לשמש כבסיס ללמידה, ניסויים ותירגול של:

- Prompt Engineering
- עבודה עם מודלים של שפה
- ארגון קוד לפרויקטי AI
- RAG
- אינטגרציה עם APIים חיצוניים

## מבנה הפרויקט

```text
rag/                  # ניסויים ופרויקטים בתחום RAG
text_to_commands/     # פרויקט Text to Commands
```

## הערה

זהו מאגר לימודי, מטרתו להציג רעיונות, קוד ודוגמאות בצורה פשוטה ונגישה למי שמתחיל ללמוד AI.
